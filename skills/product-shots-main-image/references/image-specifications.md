# Amazon US Image Specifications

Source: Amazon Product image guide, retrieved 2026-07-13:
https://sellercentral.amazon.com/gp/help/external/G1881

## Upload requirements

- Supported: JPEG, TIFF, PNG, and non-animated GIF. JPEG is recommended.
- Longest side: minimum 500 pixels and maximum 10,000 pixels.
- Use a clear image without pixelation or jagged edges.

The clothing guide recommends at least 1,000 pixels in height or width to
activate zoom. Treat 1,000+ pixels as a useful quality/zoom target, not the
universal minimum upload dimension.

## Canvas guidance

- Use 1:1 for a general catalog presentation when no category rule or design
  requirement calls for another ratio.
- Do not claim Amazon universally requires square main or alternate images.
- Adult clothing guidance calls for a ratio close to 3:4.
- Generate at a supported model canvas, then use `fit_canvas.py` to create the
  final planned dimensions. Use contain/pad for main images so the product is
  not clipped.

Alternate images may use the canvas best suited to the marketplace display and
content. Keep text readable on mobile, but do not present an unsourced point
size as a platform upload rule.
