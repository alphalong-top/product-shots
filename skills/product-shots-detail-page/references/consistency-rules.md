# A+ Consistency

Pass original product references to every generation. Use an accepted product
image as an additional composition reference, not as a replacement source of
truth. State invariants for geometry, materials, color, controls, labels,
packaging, included parts, and scale.

Use a coherent palette, type system, icon language, and lighting direction
across modules while allowing each selected template to serve its own content
objective.

Compare every generated artifact with the original product references and the
assembled A+ sequence. A prompt and visual reference reduce drift but do not
guarantee SKU identity. Do not use unsourced conversion-lift percentages as
evidence for module selection.
