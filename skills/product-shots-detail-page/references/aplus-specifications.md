# Amazon US A+ Content Specifications

Source snapshot retrieved 2026-07-13:

- A+ Content guide: https://sellercentral.amazon.com/help/hub/reference/external/GLG4RQK2Y2RJADU4
- A+ content guidelines: https://sellercentral.amazon.com/help/hub/reference/external/GGW8U76SSNTRTBX7

Recheck the target marketplace and the dimensions displayed in A+ Content
Manager before producing final assets.

## Content types and limits

| Type | General image size in guide | Maximum modules | Available module choices |
|---|---:|---:|---:|
| Basic A+ | 970 x 300 | 5 | 14 |
| Premium A+ | 1464 x 600 | 7 | 19 |

Brand Story is separate and uses one carousel area with preformatted cards.
Do not model A+ as a mandatory eight-image suite.

The selected template box is authoritative for an individual asset. Amazon may
resize an oversized image to the maximum shown by that box, while an undersized
image is not enlarged. Use Content Manager's crop/scale controls and desktop
and mobile previews.

## Technical requirements

- Formats: JPG, BMP, or PNG.
- RGB color space; no CMYK.
- Under 2 MB per image.
- At least 72 dpi.
- No animated images, watermarks, QR codes, HTML, hyperlinks, or external
  redirects.
- Provide useful alt text for every image.

Use clear, readable text and unique A+ imagery. Avoid simply reusing the image
gallery. Current guidelines also restrict promotional/pricing language,
competitor comparisons, warranty/guarantee claims, shipping details,
time-sensitive references, and personal/contact information.

## Mobile handling

Amazon requires previewing in desktop and mobile views. The official guide used
for this snapshot does not state that every image loses a fixed outer 5%.
Reserve composition bleed pragmatically, but validate the actual selected
template preview instead of claiming a universal clipping percentage or point
size.
